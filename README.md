# MusicPlayer


Single activity Music Player for beginners.

FrontEnd - XML(Relative Layout) , Kotlin 

# Concepts used :-

Implementing SeekBar in Layout file

Thread(Runnable) and Handler() used for handling position of SeekBar with playing song.
